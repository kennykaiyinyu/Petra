Detailed Description

Lock-Free Mean Gatherer (C++20)

Leverages std::atomic<double> to allow concurrent updates without mutex overhead. This is significantly faster for high-frequency updates but provides "eventual consistency" when reading results while updates are happening.

Note: std::atomic<double> requires C++20.
