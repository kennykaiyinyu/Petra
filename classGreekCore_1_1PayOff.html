Detailed Description

template<typename Derived, typename MarketParameters>
class GreekCore::PayOff< Derived, MarketParameters >

CRTP Base Class for Payoff definitions.

Uses the Curiously Recurring Template Pattern (CRTP) to enforce the interface at compile-time, avoiding virtual function overhead for high performance.

Template Parameters
DerivedThe concrete payoff implementation.
MarketParametersThe type of market data required (e.g., double for European, std::vector<double> for Asian).
