Detailed Description

template<typename PayOffType>
class GreekCore::VanillaOption< PayOffType >

A simple wrapper bundling a PayOff strategy with an Expiry time.

Useful for passing a fully defined financial instrument (Payoff + T) to a pricing engine that expects $T$ as an argument, or for building portfolios.

Template Parameters
PayOffTypeThe concrete payoff strategy (e.g., PayOffVanilla).
